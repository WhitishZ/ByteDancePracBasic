# ByteDance iOS Practice Project - Basic

2021年4月 字节跳动 字学镜像计划iOS实践项目(基础功能部分)

> Note: On Jun 8, 2023, ByteCampus community is announced to be shut down soon. Links below may be broken. The referenced resource are backed up in the `Documents` folder.

This repository is a get-started project on Xamarin.Forms application development. For personal study use only. The tasks and main functions of this project is listed in [「字学镜像计划iOS 实践篇」实践项目指导](https://bytedancecampus1.feishu.cn/docs/doccnAPVB1EAztrvfmxk77Ww6vc).
This project is developed with Microsoft Visual Studio Community 2019 using Xamarin.Forms, and tested on an Android device.

## NuGet Package Dependencies

This project depends on the following packages:

- [SQLiteNetExtensions 2.1.0](https://www.nuget.org/packages/SQLiteNetExtensions/2.1.0?_src=template)
- [Xamarin.CommunityToolkit 1.1.0](https://www.nuget.org/packages/Xamarin.CommunityToolkit/1.1.0?_src=template)
